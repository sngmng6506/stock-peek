FAQ

Frequently asked questions

Is it really free?
Yes. No ads, no payments, no sign-up. It's a personal open-source project — the full code is on GitHub.
Windows shows a warning during install.
The app isn't code-signed (it's an indie project), so SmartScreen may warn you. Click "More info → Run anyway" to install. The source is public if you'd rather inspect or build it yourself.
Where do the quotes come from?
Korean stocks from Naver Finance, US stocks from Yahoo Finance — public data, refreshed every 5 seconds during market hours and every 60 seconds after close.
Does it work on a corporate PC?
Yes. It uses the system proxy and certificates, so it works on corporate networks. You do need permission to install software.
Is there a Mac version?
Windows 10/11 only for now. No Mac plans yet.
