개인정보 및 데이터 처리 안내

몰래주식은 회원가입이나 증권사 계좌 연결을 요구하지 않습니다. 관심 종목과 보유 정보는 기본적으로 사용자의 PC에 저장됩니다.

핵심 요약

광고 SDK나 범용 행동 분석 도구는 포함하지 않습니다. 선택 기능인 “포트폴리오 한 줄 평”을 직접 요청할 때만 무작위 기기 식별자, 계산된 포트폴리오 통계와 선택 언어가 프로젝트의 Cloudflare Worker로 전송됩니다.

사용자 PC에 저장되는 정보

다음 정보는 Electron의 electron-store를 통해 사용자 PC에 저장됩니다.

  • 관심 종목의 종목 코드와 표시 순서
  • 보유 수량과 평균 매입가
  • 패널 위치, 선택한 모니터와 좌·우 배치
  • 언어와 첫 실행 안내 확인 여부
  • 포트폴리오 한 줄 평의 일일 사용 제한을 위한 무작위 기기 식별자

시세를 가져오기 위한 네트워크 요청

앱은 공개 시장 데이터를 가져오기 위해 Stock Peek Cloudflare Worker에 요청하며, 필요한 경우 네이버 금융 또는 Yahoo Finance에 직접 요청합니다. 증권사 계정에 로그인하거나 주문을 전송하지 않습니다.

포트폴리오 한 줄 평

사용자가 이 선택 기능을 직접 실행하면 다음 값이 Stock Peek Worker로 전송됩니다.

  • 무작위로 생성된 기기 식별자
  • 앱에서 계산한 포트폴리오 통계
  • 선택한 언어

기기 식별자는 하루 사용 횟수를 제한하기 위해 사용됩니다. 앱은 증권사 계정 정보나 비밀번호를 요청하지 않습니다.

광고와 분석

데스크톱 앱에는 광고 SDK나 범용 행동 분석 도구가 포함되어 있지 않습니다.
