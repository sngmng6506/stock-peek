몰래주식은 어떤 프로그램인가요?

몰래주식(Stock Peek)은 업무 중에도 창을 전환하지 않고 관심 종목을 빠르게 확인할 수 있도록 만든 무료 Windows 데스크톱 주식 시세 위젯입니다.

한 문장으로 답하면

화면 오른쪽이나 왼쪽 끝으로 마우스를 가져가면 한국·미국 관심 종목의 시세, 최근 3개월 주가 흐름과 평가손익이 나타나는 무료 오픈소스 Windows 위젯입니다.

핵심 기능

  • 화면 가장자리로 마우스를 가져가면 패널이 나타나고, 커서를 옮기면 자동으로 숨겨집니다.
  • 코스피·코스닥과 NYSE·NASDAQ 등 한국 및 미국 주요 종목을 검색할 수 있습니다.
  • 보유 수량과 평균 매입가를 입력하면 현재 평가손익을 함께 확인할 수 있습니다.
  • 최근 3개월의 일별 가격 흐름을 작은 차트로 보여줍니다.
  • 화면 왼쪽·오른쪽과 여러 모니터 중 원하는 위치에 패널을 배치할 수 있습니다.
  • 시스템 트레이에 상주하며 윈도우 시작 시 자동 실행하도록 설정할 수 있습니다.

설치하고 사용하는 방법

  1. 설치 파일 내려받기: GitHub Releases에서 최신 `.exe` 설치 파일을 내려받아 실행합니다.
  2. 관심 종목 추가하기: 종목명이나 티커를 검색하고 원하는 종목을 추가합니다. 필요하면 보유 수량과 평균 매입가도 입력합니다.
  3. 화면 가장자리에서 확인하기: 설정한 화면 가장자리로 마우스를 가져가면 패널이 나타납니다. 커서를 다른 곳으로 옮기면 자동으로 숨겨집니다.

개인이 개발한 앱이라 코드 서명 인증서가 없어 Windows SmartScreen 경고가 표시될 수 있습니다. “추가 정보”를 누른 뒤 “실행”을 선택하면 설치할 수 있습니다.

시세 출처와 갱신 주기

한국 종목은 네이버 금융, 미국 종목은 Yahoo Finance의 공개 데이터를 사용합니다. 장중에는 약 5초마다, 장 마감 후에는 약 60초마다 시세를 확인합니다. 거래소와 데이터 제공처의 정책에 따라 실제 거래 시세보다 지연될 수 있습니다.

개인정보와 로컬 저장

증권사 계좌에 로그인하거나 주문 기능과 연결하지 않습니다. 관심 종목, 보유 수량, 평균 매입가, 언어와 패널 위치 같은 설정은 사용자의 PC에 저장됩니다.

선택 기능인 “포트폴리오 한 줄 평”을 직접 요청하면, 하루 사용 횟수를 제한하고 답변을 생성하기 위해 무작위 기기 식별자와 계산된 포트폴리오 통계가 프로젝트의 Cloudflare Worker로 전송됩니다. 자세한 내용은 개인정보 안내에서 확인할 수 있습니다.

자주 묻는 질문

몰래주식은 무료인가요?
네. 광고, 결제, 회원가입 없이 무료로 사용할 수 있으며 전체 소스 코드는 MIT 라이선스로 공개되어 있습니다.
어떤 주식을 지원하나요?
코스피와 코스닥의 한국 종목, NYSE와 NASDAQ 등 미국 주요 거래소의 주식과 ETF를 지원합니다.
회사 PC에서도 사용할 수 있나요?
시스템 프록시와 인증서를 사용하도록 설계되어 있지만, 회사 보안 정책과 설치 권한에 따라 사용이 제한될 수 있습니다.
Mac에서도 사용할 수 있나요?
현재는 Windows 10과 Windows 11만 지원합니다.
표시되는 시세로 투자 판단을 해도 되나요?
표시되는 시세는 공식 주문 체결 데이터나 투자 조언이 아닙니다. 실제 거래 전에 증권사에서 최신 시세를 다시 확인해야 합니다.
