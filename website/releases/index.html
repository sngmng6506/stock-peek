최신 버전과 업데이트 내역

현재 배포 중인 Windows 설치 버전과 변경 사항을 확인할 수 있습니다. 몰래주식은 GitHub Releases를 통해 배포되며 앱 안에서도 새 버전을 자동으로 확인합니다.

최신 버전 v1.6.2

GitHub Releases에서 배포 중

한국·미국 관심 종목 시세, 최근 3개월 차트, 평가손익, 멀티모니터 배치와 자동 업데이트를 지원하는 Windows 10/11용 버전입니다.

최신 설치 파일 GitHub 릴리스 보기

설치 파일이 맞는지 확인하는 방법

각 릴리스에는 가능한 경우 SHA256SUMS.txt가 함께 제공됩니다. 설치 파일의 SHA-256 해시를 계산해 이 파일에 적힌 값과 일치하는지 비교하면 다운로드 과정에서 파일이 변경되지 않았는지 확인할 수 있습니다.

  1. GitHub의 최신 릴리스 페이지에서 설치 파일과 SHA256SUMS.txt를 내려받습니다.
  2. Windows PowerShell에서 Get-FileHash .\stock-peek-setup.exe -Algorithm SHA256을 실행합니다.
  3. 출력된 해시가 SHA256SUMS.txt의 값과 같은지 확인합니다.

자동 업데이트 방식

설치된 앱은 시작할 때와 이후 약 24시간마다 GitHub Releases의 새 버전을 확인합니다. 업데이트가 있으면 자동으로 내려받고, 사용자가 설치를 선택하거나 앱을 종료할 때 적용할 수 있습니다.

지원 환경과 주의사항

  • 지원 운영체제: Windows 10, Windows 11
  • 인터넷 연결이 필요하며 회사 PC에서는 보안 정책이나 설치 권한에 따라 제한될 수 있습니다.
  • 개인 개발 앱이라 코드 서명 인증서가 없어 Windows SmartScreen 안내가 표시될 수 있습니다.
  • 표시되는 시세는 공식 주문 체결 데이터나 투자 조언이 아닙니다.

최근 릴리스

전체 변경 내역은 GitHub Releases에서 확인할 수 있습니다.
