How to verify the installer

When available, each release includes SHA256SUMS.txt. Compare the SHA-256 hash of the installer with the value in that file to confirm that the download was not modified.

  1. Download the installer and SHA256SUMS.txt from the latest GitHub release.
  2. Run Get-FileHash .\stock-peek-setup.exe -Algorithm SHA256 in Windows PowerShell.
  3. Confirm that the printed hash matches the value in SHA256SUMS.txt.

Automatic updates

The installed app checks GitHub Releases when it starts and approximately once every 24 hours afterward. When an update is available, it is downloaded and can be installed when the user chooses or when the app exits.

Requirements and notes

  • Supported operating systems: Windows 10 and Windows 11
  • An internet connection is required. Corporate security policy or install permissions may restrict use.
  • Because the independent app is not code-signed, Windows SmartScreen may display a notice.
  • Displayed quotes are not official execution data or investment advice.
